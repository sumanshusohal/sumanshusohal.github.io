## 🛠️ How to Deploy

GitHub Pages deploys automatically from the **main branch**.

If needed:

1. Go to **Settings → Pages**
2. Under **Source**, choose **Deploy from branch**
3. Select:
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Click **Save**

Your site will be available at:

https://sumanshusohal.github.io
